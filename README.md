<h2>程式簡介</h2>
<p>同樣的南台 NEWS 的 Icon 感謝 SHXJ 製作</p>
<p>編寫語言 Python 3</p>
<p>介面使用 Tkinter</p>
<p>程式打包 Pyinstaller<p>
<p>爬蟲和字串處理</p>
<p>沒有華麗的介面</p>
<p>簡單的程式</p>

<h2>程式提供功能</h2>

1. 多頁顯示 ( 第1頁至第4頁 )
2. 單頁顯示 ( 選項包含第1頁 ~ 第10頁 )
3. 標題搜尋功能
<p>列表皆由公布日期按新至舊排序</p>
<p>點擊兩下該列可使用 " 系統預設瀏覽器 " 開啟連結</p>
<p>若要手動清空列表點擊 " 清空列表 " 按鈕, <s>這功能有點多餘</s></p>
<p>若要退出可以點擊 " Click & Quit " 按鈕 , 或點擊 X 關閉程式</p>
